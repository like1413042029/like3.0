4. 作业提示

(1). 定义
•字母: A-Z, a-z.
•字母数字: A-Z, a-z, 0-9.
•分隔符: 非字母数字
•单词:

•包含有4个或4个以上的字母
•单词由分隔符分开
•如果一个字符串包含_非_字母数字，则不是单词
•单词大小写不敏感，例如 “file”、“FILE”和“File”可以看作同一个单词
•单词必须是字母开头，“file123”是单词，“123file”不是单词

(2). 示例

输入
Word is case insensitive, i.e. “file”, “FILE” and “File” are considered the same word.

输出
file: 3
word: 2
case: 1
considered: 1
insensitive: 1
same: 1
